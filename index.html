Safari Clip Bookmarklet

Safari Clip
Once set up, you'll have a "Stash It" bookmark in Safari. Tap it on any page and the title + URL is instantly saved to Stash.
Setup (one-time)
1
Copy the code below — tap it to select, then copy.

2
In Safari, bookmark any page (Share → Add Bookmark). Name it "Stash It".
In Safari, go to any website and tap the Share button (↑ box with arrow at the bottom) → tap "Add Bookmark" → name it "Stash It" → tap Save.
3
Open Bookmarks, long-press "Stash It" → Edit. Clear the URL field and paste the copied code. Save.
Open Bookmarks in Safari → find "Stash It" → swipe left → tap Edit.

In the URL field: select all → delete → paste the code from Step 1.

Tap Done.
4
Done! Browse anywhere, open Bookmarks, tap "Stash It" — the page is saved.
Test it — browse to any page, open Bookmarks, tap "Stash It". You should see "Page stashed ✓" in Stash.
💡 Pro tip
In Safari Settings → enable Show Favorites Bar, then move "Stash It" to Favorites for one-tap clipping while browsing.
⚠️ Note: Keep Stash open in Safari (not the home screen icon) for clipping to work. Safari and the home screen app store data separately on iOS — this is a known limitation we're working around.
